# Finish Goods Warehouse (Склад готовой продукции).
## Панель форма комплектов для работы с формами заготовки.
### На данном этапе будут 3 сервиса:
    - web_fgw_admin     - Панель администратора (ведение логирования в программе, хранение истории операции)
    - web_fgw_aforms   -> Панель для работы с формами загатовки (формы для производства бутылок)
    - web_fgw           - Панель для работы со складом (хранение п\п, продажа, приход, перемещение, списание, пересортица)
